
# Selected Topics: Python for Machine Learning and Business

## 1. Why Python for Business and ML

Python is a leading programming language in business and machine learning because it balances simplicity with powerful capabilities.

### Key Benefits
- Easy to learn and use
- Rapid prototyping and development
- Large ecosystem of libraries
- Strong community and industry adoption

### Common Libraries
- pandas, NumPy – data manipulation
- matplotlib, seaborn – data visualization
- scikit-learn – machine learning
- TensorFlow, PyTorch – deep learning

---

## 2. Writing Your First Python Program

Python programs are readable and beginner-friendly.

### Example
```python
print("Hello, Python!")
```

### Concepts Introduced
- `print()` function
- String literals
- Running Python scripts

### Notes
- Python uses indentation instead of braces
- No need for semicolons
- Code is executed line by line
