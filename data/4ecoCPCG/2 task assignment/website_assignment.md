# Introduction to Python

## Introduction
Python is a high-level, interpreted programming language known for its simplicity and readability. It is widely used in education, business, and economics because it helps in analysing data and solving real-world problems.

## Key Concepts
- High-level programming language
- Easy to read and write
- Free and open-source
- Useful for data analysis and research

## Simple Syntax Examples

print("Hello, Python")

a = 10
b = 5
print(a + b)

## Uses of Python in Economics
Python is used in economics to analyse large datasets, perform statistical and econometric analysis, forecast economic trends, study consumer behaviour, and evaluate public policies.

## Real-Life or Business Use Case
In real life, Python is used by economists, banks, government organisations, and research institutions to analyse GDP, inflation, unemployment, and market trends, helping in informed decision-making.

## Advantages
- Easy to learn
- Saves time
- Large number of libraries
- Free to use

## Limitations
- Slower than some other languages
- Not suitable for mobile apps
- Needs external libraries

## Summary
Python is a simple yet powerful programming language that is widely used in economics and business analysis. Its easy syntax and strong data-handling capabilities make it an important tool for students and researchers.
